example-go-slackbot
===============================

## Application Environment
- `SLACK_BOT_TOKEN`
- `DOCOMO_APIKEY`
- `CSE_CX`
- `CSE_APIKEY`
- `REDISTOGO_URL`: `redis://<id>:<password>@<host>:<port>` https://redistogo.com/

## Wercker Environment
- DockerHub push
    - `DOCKER_HUB_USERNAME`
    - `DOCKER_HUB_PASSWORD`
- Arukas restart
    - `ARUKAS_JSON_API_TOKEN`
    - `ARUKAS_JSON_API_SECRET`
    - `ARUKAS_CONTAINER_ID`
